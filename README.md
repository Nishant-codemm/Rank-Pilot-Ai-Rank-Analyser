# 📊 AI SEO Rank Tracker

An AI-powered SEO tool designed to analyze website rankings, keywords, and overall performance metrics. 

## ✨ Key Features
- **Keyword Tracking:** Monitor website rankings for specific target keywords.
- **AI-Powered Insights:** Get actionable SEO recommendations.
- **Performance Metrics:** Analyze page speed and Core Web Vitals.

## 🛠️ Project Structure
- **/Client:** Built with React/Vite.
- **/server:** Built with Node.js/Express.

## 🚀 Getting Started
1. Clone the repository: `git clone [YOUR_REPO_LINK]`
2. Install dependencies: `npm install`
3. Start the project: `npm run dev`
